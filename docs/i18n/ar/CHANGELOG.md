# سجل التغييرات

الصيغة: [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) · إصدار دلالي (semantic versioning).

## [1.0.0] — 2026-07-07

الإصدار العلني الأول.

### المنتج

- **بروكسي ضغط السياق كصورة**: يعيد كتابة الأجزاء الضخمة من كل طلب LLM
  (system prompt، مستندات الأدوات، التاريخ القديم، مخرجات الأدوات الكبيرة) إلى صفحات
  PNG كثيفة بعمق 1-بت قبل أن تغادر جهازك. خادم Node محلي ومضيف
  Cloudflare Workers.
- **رياضيات فوترة دقيقة لكل مزوّد** (`src/core/`): رقاقات Anthropic بحجم 28 بكسل +
  3–4 رموز/كتلة كتكلفة إضافية (قياس خاص، انحراف صفري)، معادلات OpenAI وGemini
  مدقَّقة مقابل المستندات الرسمية. مُصدَّرة عند جذر الحزمة
  (`anthropicImageTokens`، `resolveAnthropicVisionTier`، حدود المستويات).
- **إعداد رندر إنتاجي مقيس**: أطلس رموز 1-بت كثيف (بلا تنعيم حواف/anti-aliasing)،
  صفحات المستوى القياسي — كل خيار مدعوم بإيصال قياس
  في `benchmarks/*/results/`.
- **حزم القياس** (`benchmarks/`): billing-sweep (محاسبة الرموز) و
  density-frontier (حدود دقة القراءة عبر النماذج/الكثافات)، قابلة لإعادة التشغيل
  عبر API، أو OpenRouter، أو CLI الخاص بـ Claude Code، أو عبر OmniRoute
  (`--via-omniroute`).
- **إعادة محاولة عند الرفض**: مستشعر SSE/JSON يعيد إرسال الطلب الأصلي عندما يرفض نموذج
  الصفحة المرندرة (مفتاح إيقاف طوارئ `retryRefusalWithOriginal`).
- **ذاكرة تخزين مؤقت LRU للرندر** لضمان صفحات حتمية.
- **محرك OmniRoute**: يعمل كمحرك ضغط `omniglyph` داخل
  [OmniRoute](https://github.com/diegosouzapw/OmniRoute) (وضع منفرد وخط أنابيب مكدّس)،
  مع بوابات فشل مغلق ومحاسبة رموز واعية بالصور.

### الأرقام (كلها قابلة لإعادة الإنتاج)

- رندر واجهة عيّنة: 1015 حرفًا → PNG بأبعاد 438×120، 254 → 84 رمزًا (**توفير 66.9%**).
- الصفحة القياسية 1568×728 = 1456 رمز صورة بغضّ النظر عن كمية النص التي تحملها.
- Claude يقرأ الصفحات الكثيفة أحادية البت بدقة 100% عند كثافة الإنتاج؛ Opus 4.8 يقرأ
  77–87% عند 10×16.

### قرارات سلبية (مقيسة، وليست آراء)

- **مستوى الدقة العالية فخّ فوترة**: صفحة 1928² تُحاسَب حسب حجمها الفعلي (WYSIWYG)
  لكن المُرمِّز لا يستقبل الدقة الكاملة — كلا المستويين يرندران صفحات قياسية.
- **GPT-5.5 مرفوض**: 0/60 قراءة للشريط الكثيف و~40× تضخّم في الإكمال
  مقارنةً بضابط النص.
- **gpt-4o-mini لم يُصوَّر أبدًا** (حد أدنى 2833/5667 رمزًا يجعله غير مربح).
- **Gemini 2.5-flash يختلق** بدل أن يمتنع عن الإجابة على الصفحات الكثيفة
  (0/26) — في انتظار إعادة اختبار بحصة مدفوعة.
