# Архитектура

Едностранична карта на кодовата база.

## Тръбопровод на заявките

```
клиент (Claude Code / всяко SDK)
  │  ANTHROPIC_BASE_URL=http://127.0.0.1:47821
  ▼
src/node.ts · src/worker.ts        хостове (node:http / Cloudflare Workers)
  ▼
src/core/proxy.ts                  единен fetch handler по Web стандарт:
  │                                маршрутизация, препращане на автентикация, count_tokens
  │                                контрафактуал, събития за употреба/телеметрия
  ▼
src/core/transform.ts              ГЛАВНИЯТ тръбопровод (път за Anthropic):
  │   1. парсиране на тялото, разрешаване на нивото за визия от модела
  │   2. врата за рентабилност — точна цена на изображение срещу текст
  │   3. конвертиране: статичен slab · големи tool_results · свита история
  │   4. сглобяване обратно, запазвайки котвите на клиентския cache_control
  ▼
upstream API (api.anthropic.com / api.openai.com)
```

## Таксуване (точно, измерено)

| модул | доставчик | модел |
|---|---|---|
| `src/core/anthropic-vision.ts` | Anthropic | пач-размери 28 px + 4/блок, лимити за преоразмеряване на ниво; геометрия на страницата (и двете нива рендират стандартната страница 1568×728 — нивото с висока резолюция е капан при таксуването, вижте [BENCHMARKS](../benchmarks/BENCHMARKS.md)) |
| `src/core/gpt-model-profiles.ts` + `openai.ts` | OpenAI | режими пач/тайл на модел, `detail` по профил, геометрия на лентата |
| `src/core/gemini-model-profiles.ts` | Google | формула за тайлове (`floor(min/1.5)` единица за изрязване) + фиксирани цени за `media_resolution` |

## Рендиране

- `src/core/render.ts` — текст → PNG чрез вграден атлас с глифи (Spleen 5×8 +
  резервен Unifont), reflow с `↵` sentinel символи за нов ред, 1-битов атлас в
  производството (измерено по-добро от AA при Fable).
- `src/core/render-cache.ts` — LRU мемоизация на детерминистични рендери
  (иначе статичният slab + замразените части от историята се рендират наново при всяка заявка).
- `src/core/history.ts` — свива старите ходове в append-only замразени части
  от изображения, които остават идентични по байт, така че кешът на промптите продължава да улучва.
- `src/core/png.ts` — минимален детерминистичен PNG енкодер (без нативни зависимости).

## Предпазни мерки

- Бял списък на модели (`src/core/applicability.ts`): само модели, преминали
  бенчмарка за четене, се изобразяват; всичко останало преминава непокътнато, байт по байт.
- Точните до байт стойности (SHA-та, ID-та) пътуват като текст в факт лист до изображението
  (`src/core/factsheet.ts`); възстановими оригинали чрез `emitRecoverable`.
- Нативните типизирани инструменти (`type !== 'custom'`) никога не се пренаписват (защита срещу 400).
