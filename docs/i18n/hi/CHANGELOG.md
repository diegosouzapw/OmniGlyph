# Changelog

फ़ॉर्मैट: [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) · सिमेंटिक वर्ज़निंग।

## [1.0.0] — 2026-07-07

पहला सार्वजनिक रिलीज़।

### उत्पाद

- **संदर्भ-को-छवि संपीड़न प्रॉक्सी**: हर LLM अनुरोध के भारी-भरकम हिस्सों
  (सिस्टम प्रॉम्प्ट, टूल दस्तावेज़, पुराना इतिहास, बड़े टूल आउटपुट) को आपकी
  मशीन छोड़ने से पहले सघन 1-bit PNG पेजों में फिर से लिखता है। स्थानीय Node
  सर्वर और Cloudflare Workers होस्ट।
- **प्रति-प्रोवाइडर सटीक बिलिंग गणित** (`src/core/`): Anthropic 28px पैच +
  3–4 टोकन/ब्लॉक ओवरहेड (अपना स्वीप, शून्य अवशिष्ट), OpenAI और Gemini
  फ़ॉर्मूले आधिकारिक दस्तावेज़ों के विरुद्ध ऑडिट किए गए। पैकेज रूट पर
  एक्सपोर्ट किए गए (`anthropicImageTokens`, `resolveAnthropicVisionTier`,
  टियर कैप)।
- **मापा गया प्रोडक्शन रेंडर कॉन्फ़िग**: सघन 1-bit ग्लिफ़ एटलस (कोई
  एंटी-अलियासिंग नहीं), स्टैंडर्ड-टियर पेज — हर विकल्प के पीछे
  `benchmarks/*/results/` में एक बेंचमार्क रसीद।
- **बेंचमार्क हार्नेस** (`benchmarks/`): billing-sweep (टोकन लेखांकन) और
  density-frontier (मॉडल/घनत्वों में पठन-सटीकता की सीमा), API, OpenRouter,
  Claude Code CLI, या OmniRoute (`--via-omniroute`) के ज़रिए दोबारा चलाने
  योग्य।
- **अस्वीकृति पुनर्प्रयास**: SSE/JSON स्निफ़र मूल अनुरोध को दोहराता है जब
  कोई मॉडल रेंडर किए गए पेज को अस्वीकार करता है (किल स्विच
  `retryRefusalWithOriginal`)।
- निर्धारक पेजों के लिए **LRU रेंडर कैश**।
- **OmniRoute इंजन**: [OmniRoute](https://github.com/diegosouzapw/OmniRoute)
  में `omniglyph` संपीड़न इंजन के रूप में शिप होता है (सिंगल मोड और स्टैक्ड
  पाइपलाइन), फेल-क्लोज़्ड गेट्स और छवि-जागरूक टोकन लेखांकन के साथ।

### आँकड़े (सभी पुनरुत्पादनीय)

- नमूना UI रेंडर: 1015 अक्षर → 438×120 PNG, 254 → 84 टोकन (**66.9% बचत**)।
- स्टैंडर्ड पेज 1568×728 = 1456 छवि टोकन, चाहे उसमें कितना भी टेक्स्ट हो।
- Claude प्रोडक्शन घनत्व पर सघन 1-bit पेज 100% पढ़ता है; Opus 4.8 10×16 पर
  77–87% पढ़ता है।
