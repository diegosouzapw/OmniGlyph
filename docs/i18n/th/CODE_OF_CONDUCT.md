# จรรยาบรรณของผู้มีส่วนร่วม (Contributor Covenant)

## คำมั่นสัญญาของเรา

ในฐานะสมาชิก ผู้มีส่วนร่วม และผู้นำ เราให้คำมั่นว่าจะทำให้การมีส่วนร่วมใน
ชุมชนของเราเป็นประสบการณ์ที่ปราศจากการล่วงละเมิดสำหรับทุกคน โดยไม่คำนึงถึงอายุ ขนาดร่างกาย
ความพิการที่มองเห็นหรือมองไม่เห็น เชื้อชาติ ลักษณะทางเพศ อัตลักษณ์และการแสดงออก
ทางเพศ ระดับประสบการณ์ การศึกษา สถานะทางเศรษฐกิจและสังคม
สัญชาติ รูปลักษณ์ภายนอก เชื้อชาติ ศาสนา หรืออัตลักษณ์ทางเพศ
และรสนิยมทางเพศ

เราให้คำมั่นว่าจะกระทำและมีปฏิสัมพันธ์ในรูปแบบที่ส่งเสริมชุมชนที่เปิดกว้าง ต้อนรับ
หลากหลาย ครอบคลุม และมีสุขภาพดี

## มาตรฐานของเรา

ตัวอย่างพฤติกรรมที่ส่งเสริมสภาพแวดล้อมเชิงบวกสำหรับ
ชุมชนของเรา ได้แก่:

- แสดงความเห็นอกเห็นใจและความเมตตาต่อผู้อื่น
- เคารพความคิดเห็น มุมมอง และประสบการณ์ที่แตกต่าง
- ให้และยอมรับข้อเสนอแนะเชิงสร้างสรรค์ด้วยความสง่างาม
- ยอมรับความรับผิดชอบและขอโทษผู้ที่ได้รับผลกระทบจากความผิดพลาดของเรา
  และเรียนรู้จากประสบการณ์นั้น
- มุ่งเน้นสิ่งที่ดีที่สุดไม่ใช่แค่สำหรับตัวเราในฐานะปัจเจกบุคคล แต่สำหรับ
  ชุมชนโดยรวม

ตัวอย่างพฤติกรรมที่ยอมรับไม่ได้ ได้แก่:

- การใช้ภาษาหรือภาพที่มีนัยทางเพศ และความสนใจหรือ
  การรุกล้ำทางเพศในทุกรูปแบบ
- การกลั่นแกล้ง ดูหมิ่นหรือแสดงความคิดเห็นดูถูก และการโจมตีส่วนบุคคลหรือทางการเมือง
- การล่วงละเมิดในที่สาธารณะหรือส่วนตัว
- การเผยแพร่ข้อมูลส่วนตัวของผู้อื่น เช่น ที่อยู่ทางกายภาพหรืออีเมล
  โดยไม่ได้รับอนุญาตอย่างชัดแจ้ง
- พฤติกรรมอื่นใดที่อาจถือได้อย่างสมเหตุสมผลว่าไม่เหมาะสมใน
  สภาพแวดล้อมแบบมืออาชีพ

## ความรับผิดชอบในการบังคับใช้

ผู้นำชุมชนมีหน้าที่รับผิดชอบในการชี้แจงและบังคับใช้มาตรฐานของเราเกี่ยวกับ
พฤติกรรมที่ยอมรับได้ และจะดำเนินการแก้ไขที่เหมาะสมและยุติธรรมใน
การตอบสนองต่อพฤติกรรมใดๆ ที่พวกเขาเห็นว่าไม่เหมาะสม คุกคาม ก้าวร้าว
หรือเป็นอันตราย

ผู้นำชุมชนมีสิทธิและความรับผิดชอบในการลบ แก้ไข หรือปฏิเสธ
ความคิดเห็น commit โค้ด การแก้ไข wiki ปัญหา และการมีส่วนร่วมอื่นๆ ที่
ไม่สอดคล้องกับจรรยาบรรณนี้ และจะสื่อสารเหตุผลของการตัดสินใจ
ในการกลั่นกรองเมื่อเหมาะสม

## ขอบเขต

จรรยาบรรณนี้บังคับใช้ในพื้นที่ชุมชนทั้งหมด และยังบังคับใช้เมื่อ
บุคคลใดเป็นตัวแทนของชุมชนอย่างเป็นทางการในพื้นที่สาธารณะ
ตัวอย่างของการเป็นตัวแทนของชุมชนของเรา ได้แก่ การใช้ที่อยู่อีเมลที่เป็นทางการ
การโพสต์ผ่านบัญชีโซเชียลมีเดียที่เป็นทางการ หรือการทำหน้าที่เป็นตัวแทนที่ได้รับการแต่งตั้ง
ในกิจกรรมออนไลน์หรือออฟไลน์

## การบังคับใช้

กรณีของพฤติกรรมที่เป็นการล่วงละเมิด คุกคาม หรือไม่เป็นที่ยอมรับในรูปแบบอื่นๆ อาจ
รายงานไปยังผู้นำชุมชนที่รับผิดชอบในการบังคับใช้โดยการเปิดคำแนะนำ
ด้านความปลอดภัยแบบส่วนตัวที่
<https://github.com/diegosouzapw/OmniGlyph/security/advisories/new>
หรือโดยการส่งอีเมลถึงผู้ดูแลโดยตรงที่ diegosouza.pw@outlook.com
สำหรับเหตุการณ์ที่เกี่ยวข้องกับความปลอดภัย โปรดดู [`SECURITY.md`](SECURITY.md)
ข้อร้องเรียนทั้งหมดจะได้รับการตรวจสอบและสอบสวนอย่างรวดเร็วและเป็นธรรม

ผู้นำชุมชนทุกคนมีข้อผูกพันที่จะเคารพความเป็นส่วนตัวและความปลอดภัยของ
ผู้รายงานเหตุการณ์ใดๆ

## แนวทางการบังคับใช้

ผู้นำชุมชนจะปฏิบัติตามแนวทางผลกระทบต่อชุมชนเหล่านี้ในการพิจารณา
ผลที่ตามมาสำหรับการกระทำใดๆ ที่พวกเขาเห็นว่าละเมิดจรรยาบรรณนี้:

### 1. การแก้ไข

**ผลกระทบต่อชุมชน**: การใช้ภาษาที่ไม่เหมาะสมหรือพฤติกรรมอื่นที่ถือว่า
ไม่เป็นมืออาชีพหรือไม่เป็นที่ต้อนรับในชุมชน

**ผลที่ตามมา**: คำเตือนเป็นลายลักษณ์อักษรแบบส่วนตัวจากผู้นำชุมชน โดยให้ความชัดเจน
เกี่ยวกับลักษณะของการละเมิดและคำอธิบายว่าทำไม
พฤติกรรมนั้นจึงไม่เหมาะสม อาจมีการขอให้ขอโทษต่อสาธารณะ

### 2. คำเตือน

**ผลกระทบต่อชุมชน**: การละเมิดผ่านเหตุการณ์เดียวหรือชุดของ
การกระทำ

**ผลที่ตามมา**: คำเตือนพร้อมผลที่ตามมาสำหรับพฤติกรรมที่ยังคงดำเนินต่อไป ไม่มี
ปฏิสัมพันธ์กับบุคคลที่เกี่ยวข้อง รวมถึงปฏิสัมพันธ์ที่ไม่พึงประสงค์กับ
ผู้ที่บังคับใช้จรรยาบรรณ เป็นระยะเวลาหนึ่ง ซึ่ง
รวมถึงการหลีกเลี่ยงการปฏิสัมพันธ์ในพื้นที่ชุมชนตลอดจนช่องทางภายนอก
เช่น โซเชียลมีเดีย การละเมิดเงื่อนไขเหล่านี้อาจนำไปสู่การแบนชั่วคราวหรือ
ถาวร

### 3. การแบนชั่วคราว

**ผลกระทบต่อชุมชน**: การละเมิดมาตรฐานชุมชนอย่างร้ายแรง รวมถึง
พฤติกรรมที่ไม่เหมาะสมอย่างต่อเนื่อง

**ผลที่ตามมา**: การแบนชั่วคราวจากปฏิสัมพันธ์หรือการสื่อสารสาธารณะทุกรูปแบบกับ
ชุมชนเป็นระยะเวลาหนึ่ง ไม่อนุญาตให้มีปฏิสัมพันธ์สาธารณะหรือ
ส่วนตัวกับบุคคลที่เกี่ยวข้อง รวมถึงปฏิสัมพันธ์ที่ไม่พึงประสงค์
กับผู้ที่บังคับใช้จรรยาบรรณ ในช่วงเวลานี้
การละเมิดเงื่อนไขเหล่านี้อาจนำไปสู่การแบนถาวร

### 4. การแบนถาวร

**ผลกระทบต่อชุมชน**: การแสดงรูปแบบของการละเมิดมาตรฐานชุมชน
รวมถึงพฤติกรรมที่ไม่เหมาะสมอย่างต่อเนื่อง การล่วงละเมิดบุคคล
หรือความก้าวร้าวต่อหรือการดูหมิ่นกลุ่มบุคคล

**ผลที่ตามมา**: การแบนถาวรจากปฏิสัมพันธ์สาธารณะทุกรูปแบบภายใน
ชุมชน
