## ความเสี่ยงที่ยังเปิดอยู่

- การปฏิเสธของ Fable หลัง redeploy ในบริบทที่แปลงเป็นภาพ (upstream #37) — บรรเทาก่อนเปิดใช้เป็นค่าเริ่มต้นใน OmniRoute
- การเก็งกำไรราคา: ถ้า Anthropic ปรับราคาการมองเห็นใหม่ การประหยัดจะเปลี่ยน — ค่าตรงข้ามต่อคำขอ (`count_tokens`) คือแนวป้องกัน
- OpenAI: การวัดผลของชุมชน (PageWatch) พบว่าโทเคนผลลัพธ์เพิ่มขึ้นและ latency สูงขึ้น 2 เท่า — ต้องวัดต่อผู้ให้บริการก่อนเปิดใช้งาน
