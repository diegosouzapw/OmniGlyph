# Contributor Covenant Code of Conduct

## અમારું વચન

સભ્યો, contributors, અને leaders તરીકે અમે અમારા સમુદાયમાં ભાગીદારીને દરેક
માટે ઉંમર, શરીરનું કદ, દેખાતી કે ન દેખાતી અક્ષમતા, વંશીયતા, જાતિ લક્ષણો,
gender identity અને expression, અનુભવનું સ્તર, શિક્ષણ, સામાજિક-આર્થિક
સ્થિતિ, રાષ્ટ્રીયતા, વ્યક્તિગત દેખાવ, જાતિ, ધર્મ, અથવા જાતીય ઓળખ અને
અભિગમને ધ્યાનમાં લીધા વગર harassment-free અનુભવ બનાવવાનું વચન આપીએ છીએ.

અમે ખુલ્લા, આવકારદાયક, વૈવિધ્યસભર, સમાવેશી, અને સ્વસ્થ સમુદાયમાં યોગદાન
આપતી રીતે કાર્ય કરવા અને interact કરવાનું વચન આપીએ છીએ.

## અમારા ધોરણો

અમારા સમુદાય માટે સકારાત્મક વાતાવરણમાં યોગદાન આપતા વર્તનના ઉદાહરણોમાં
સામેલ છે:

- અન્ય લોકો પ્રત્યે empathy અને દયા દર્શાવવી
- અલગ-અલગ મંતવ્યો, દૃષ્ટિકોણો, અને અનુભવો પ્રત્યે આદર રાખવો
- constructive feedback આપવું અને ગ્રેસફૂલી સ્વીકારવું
- અમારી ભૂલોથી પ્રભાવિત થયેલા લોકો પ્રત્યે જવાબદારી સ્વીકારવી અને માફી
  માંગવી, અને અનુભવમાંથી શીખવું
- ફક્ત વ્યક્તિઓ તરીકે અમારા માટે જ નહીં, પણ સંપૂર્ણ સમુદાય માટે શ્રેષ્ઠ શું
  છે તેના પર ધ્યાન કેન્દ્રિત કરવું

અસ્વીકાર્ય વર્તનના ઉદાહરણોમાં સામેલ છે:

- જાતીય ભાષા અથવા ઈમેજરીનો ઉપયોગ, અને કોઈપણ પ્રકારનું જાતીય ધ્યાન અથવા
  advances
- Trolling, અપમાનજનક કે ડિરોગેટરી ટિપ્પણીઓ, અને વ્યક્તિગત કે રાજકીય હુમલા
- જાહેર અથવા ખાનગી harassment
- અન્યની ખાનગી માહિતી, જેમ કે physical અથવા email address, તેમની સ્પષ્ટ
  પરવાનગી વગર પ્રકાશિત કરવી
- professional સેટિંગમાં યોગ્ય રીતે અયોગ્ય ગણી શકાય તેવું અન્ય વર્તન

## Enforcement જવાબદારીઓ

Community leaders સ્વીકાર્ય વર્તનના અમારા ધોરણોને સ્પષ્ટ કરવા અને લાગુ
કરવા માટે જવાબદાર છે અને તેઓ જે વર્તનને અયોગ્ય, ધમકીભર્યું, અપમાનજનક, અથવા
હાનિકારક ગણે તેના જવાબમાં યોગ્ય અને ન્યાયી corrective action લેશે.

Community leaders ને comments, commits, code, wiki edits, issues, અને અન્ય
યોગદાન કે જે આ Code of Conduct સાથે align નથી તેને દૂર કરવાનો, edit કરવાનો,
અથવા reject કરવાનો અધિકાર અને જવાબદારી છે, અને જ્યારે યોગ્ય હોય ત્યારે
moderation નિર્ણયો માટેના કારણો જણાવશે.

## અવકાશ

આ Code of Conduct બધા સમુદાય spaces ની અંદર લાગુ પડે છે, અને જ્યારે કોઈ
વ્યક્તિ સત્તાવાર રીતે જાહેર spaces માં સમુદાયનું પ્રતિનિધિત્વ કરી રહી હોય
ત્યારે પણ લાગુ પડે છે. અમારા સમુદાયનું પ્રતિનિધિત્વ કરવાના ઉદાહરણોમાં
સત્તાવાર e-mail address વાપરવું, સત્તાવાર social media account દ્વારા
પોસ્ટ કરવું, અથવા ઓનલાઇન કે ઓફલાઇન ઇવેન્ટમાં નિયુક્ત પ્રતિનિધિ તરીકે કાર્ય
કરવું સામેલ છે.

## Enforcement

અપમાનજનક, harassing, અથવા અન્યથા અસ્વીકાર્ય વર્તનની ઘટનાઓ enforcement માટે
જવાબદાર community leaders ને
<https://github.com/diegosouzapw/OmniGlyph/security/advisories/new> પર
ખાનગી security advisory ખોલીને અથવા maintainer ને
diegosouza.pw@outlook.com પર email કરીને જાણ કરી શકાય છે.
Security-sensitive incidents માટે, જુઓ [`SECURITY.md`](SECURITY.md).
બધી ફરિયાદોની ઝડપથી અને ન્યાયપૂર્વક સમીક્ષા અને તપાસ કરવામાં આવશે.

બધા community leaders કોઈપણ ઘટનાના reporter ની ગોપનીયતા અને સુરક્ષાનું
સન્માન કરવા બંધાયેલા છે.

## Enforcement Guidelines

Community leaders આ Code of Conduct ના ઉલ્લંઘનમાં તેઓ જે કોઈપણ action ને
ગણે તેના પરિણામો નક્કી કરવામાં આ Community Impact Guidelines નું પાલન
કરશે:

### 1. Correction

**Community Impact**: અયોગ્ય ભાષાનો ઉપયોગ અથવા સમુદાયમાં unprofessional
અથવા unwelcome ગણાતું અન્ય વર્તન.

**Consequence**: community leaders તરફથી ખાનગી, લેખિત ચેતવણી, જે ઉલ્લંઘનની
પ્રકૃતિ વિશે સ્પષ્ટતા અને વર્તન શા માટે અયોગ્ય હતું તેની સમજૂતી આપે છે.
જાહેર માફીની વિનંતી થઈ શકે છે.

### 2. Warning

**Community Impact**: એક જ ઘટના અથવા actions ની શ્રેણી દ્વારા ઉલ્લંઘન.

**Consequence**: ચાલુ વર્તન માટે પરિણામો સાથેની ચેતવણી. Code of Conduct
લાગુ કરનારાઓ સાથે unsolicited interaction સહિત, સામેલ લોકો સાથે કોઈ
interaction નહીં, ચોક્કસ સમયગાળા માટે. આમાં community spaces તેમજ social
media જેવા external channels માં interactions ટાળવાનો સમાવેશ થાય છે. આ
શરતોનું ઉલ્લંઘન કરવાથી temporary અથવા permanent ban થઈ શકે છે.

### 3. Temporary Ban

**Community Impact**: સતત અયોગ્ય વર્તન સહિત, community standards નું
ગંભીર ઉલ્લંઘન.

**Consequence**: ચોક્કસ સમયગાળા માટે સમુદાય સાથે કોઈપણ પ્રકારના
interaction અથવા જાહેર communication થી temporary ban. Code of Conduct
લાગુ કરનારાઓ સાથે unsolicited interaction સહિત, સામેલ લોકો સાથે આ સમયગાળા
દરમિયાન કોઈ જાહેર કે ખાનગી interaction મંજૂર નથી. આ શરતોનું ઉલ્લંઘન
કરવાથી permanent ban થઈ શકે છે.

### 4. Permanent Ban

**Community Impact**: સતત અયોગ્ય વર્તન, કોઈ વ્યક્તિની harassment, અથવા
વ્યક્તિઓના classes પ્રત્યે aggression અથવા disparagement સહિત, community
standards ના ઉલ્લંઘનની pattern દર્શાવવી.

**Consequence**: સમુદાયની અંદર કોઈપણ પ્રકારના જાહેર interaction થી
permanent ban.
