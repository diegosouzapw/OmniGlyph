# יומן שינויים

פורמט: [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) · ניהול גרסאות סמנטי.

## [1.0.0] — 2026-07-07

מהדורה ציבורית ראשונה.

### המוצר

- **פרוקסי דחיסת הקשר-כתמונה**: כותב מחדש את החלקים הגדושים של כל בקשת LLM
  (הנחיית מערכת, תיעוד כלים, היסטוריה ישנה, פלטי כלים גדולים) לעמודי PNG
  צפופים בעלי 1-ביט לפני שהם עוזבים את המחשב שלכם. שרת Node מקומי ומארח
  Cloudflare Workers.
- **מתמטיקת חיוב מדויקת לכל ספק** (`src/core/`): טלאים של 28 פיקסלים ב-Anthropic +
  תקורה של 3–4 טוקנים לבלוק (מדידה עצמית, שארית אפס), נוסחאות OpenAI ו-Gemini
  שבוקרו מול התיעוד הרשמי. מיוצא בשורש החבילה
  (`anthropicImageTokens`, `resolveAnthropicVisionTier`, תקרות רמה).
- **תצורת רינדור ייצור נמדדת**: אטלס גליפים 1-ביט צפוף (ללא
  anti-aliasing), עמודי רמה סטנדרטית — כל בחירה מגובה בקבלת בנצ'מרק
  ב-`benchmarks/*/results/`.
- **רתמות בנצ'מרק** (`benchmarks/`): billing-sweep (חשבונאות טוקנים) ו-
  density-frontier (חזית דיוק קריאה על פני מודלים/צפיפויות), ניתנות להרצה מחדש
  דרך API, OpenRouter, Claude Code CLI, או דרך OmniRoute
  (`--via-omniroute`).
- **ניסיון חוזר בסירוב**: מריח SSE/JSON שמשחזר את הבקשה המקורית כאשר מודל
  מסרב לעמוד המרונדר (מתג כיבוי `retryRefusalWithOriginal`).
- **מטמון רינדור LRU** לעמודים דטרמיניסטיים.
- **מנוע OmniRoute**: משמש כמנוע הדחיסה `omniglyph` ב-
  [OmniRoute](https://github.com/diegosouzapw/OmniRoute) (מצב בודד וצינור מוערם),
  עם שערי fail-closed וחשבונאות טוקנים מודעת-תמונה.

### המספרים (הכל ניתן לשחזור)

- דוגמת רינדור ממשק: 1015 תווים → PNG בגודל 438×120, 254 → 84 טוקנים (**66.9% נחסך**).
- עמוד סטנדרטי 1568×728 = 1456 טוקני תמונה ללא תלות בכמות הטקסט שהוא נושא.
- Claude קורא עמודים צפופים בעלי 1-ביט ב-100% ברמת צפיפות הייצור; Opus 4.8 קורא
  77–87% ב-10×16.

### החלטות שליליות (נמדדו, לא דעות)

- **רמת הרזולוציה הגבוהה היא מלכודת חיוב**: עמוד 1928² מחויב WYSIWYG אך
  המקודד אינו מקבל רזולוציה מלאה — שתי הרמות מרנדרות עמודים סטנדרטיים.
- **GPT-5.5 נדחה**: 0/60 קריאות של הרצועה הצפופה וניפוח השלמה של פי ~40
  לעומת בקרת הטקסט.
- **gpt-4o-mini מעולם לא הודמה** (רצפת 2833/5667 טוקנים הופכת זאת ללא רווחי).
- **Gemini 2.5-flash בודה** במקום להימנע על עמודים צפופים
  (0/26) — ממתין לבדיקה חוזרת עם מכסה בתשלום.
