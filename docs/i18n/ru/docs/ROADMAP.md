## Результаты A/B 2026-07-05 (через OpenRouter — НЕОКОНЧАТЕЛЬНО для геометрии, валидно для режимов отказа)

| конфигурация | verbatim | abst. | filtered | silent-wrong |
|---|---|---|---|---|
| fable std 5×8 (AA и 1-bit) | 0/30 | 0 | **30/30 content_filter** | 0 |
| fable hires 5×8 (AA) | 0/30 | **20 ILEGIVEL** | 0 | 5 (2 предсказано) |
| fable hires 5×8 (1-bit) | 0/30 | 20 | 1 | 4 (2 предсказано) |
| opus hires 10×16 | **7/9 прочитано** | 0 | 21 закончились кредиты | 2 (цифра) |

Валидные находки: (1) классификатор (issue #37) — ДОМИНИРУЮЩИЙ режим отказа
для вопросов транскрипции на стандартной странице — 100% отфильтровано — и не
срабатывает на большой странице; формулировка имеет значение. (2) Абстенция
работает: 20× ILEGIVEL против 5 конфабуляций на большой странице. (3) Opus при
10×16 читает 78% точно (n=9) против 0% исторически при 5×8 — первое прямое
свидетельство «колена» кривой. (4) Нечитаемость большой страницы через
OpenRouter указывает на транспортный RESAMPLE (стандартный тир
Bedrock/Vertex?) — решающая гипотеза для проверки на прямом API Anthropic;
A/B по геометрии остаётся ОТКРЫТЫМ до тех пор. Кредиты OpenRouter
закончились в середине плеча Opus.

resample); вердикт по WYSIWYG-геометрии всё ещё зависит от прямого API.
