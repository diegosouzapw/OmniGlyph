# Архитектура

Карта кодовой базы на одну страницу.

## Конвейер запроса

```
клиент (Claude Code / любой SDK)
  │  ANTHROPIC_BASE_URL=http://127.0.0.1:47821
  ▼
src/node.ts · src/worker.ts        хосты (node:http / Cloudflare Workers)
  ▼
src/core/proxy.ts                  единый Web-standard fetch-обработчик:
  │                                маршрутизация, проброс auth, контрфактическое
  │                                count_tokens, события usage/телеметрии
  ▼
src/core/transform.ts              ОСНОВНОЙ конвейер (путь Anthropic):
  │   1. парсинг тела, определение vision-тира по модели
  │   2. гейт рентабельности — точная стоимость изображения против текста
  │   3. конвертация: статичный slab · крупные tool_results · свёрнутая история
  │   4. сплайсинг обратно с сохранением якорей client cache_control
  ▼
upstream API (api.anthropic.com / api.openai.com)
```

## Биллинг (точный, измеренный)

| модуль | провайдер | модель |
|---|---|---|
| `src/core/anthropic-vision.ts` | Anthropic | патчи 28 px + 4/блок, лимиты ресайза по тирам; геометрия страницы (оба тира рендерят стандартную страницу 1568×728 — тир высокого разрешения — биллинговая ловушка, см. [BENCHMARKS](../benchmarks/BENCHMARKS.md)) |
| `src/core/gpt-model-profiles.ts` + `openai.ts` | OpenAI | режимы patch/tile по каждой модели, `detail` по профилю, strip-геометрия |
| `src/core/gemini-model-profiles.ts` | Google | формула tile (единица кропа `floor(min/1.5)`) + плоские стоимости `media_resolution` |

## Рендеринг

- `src/core/render.ts` — текст → PNG через запечённый атлас глифов (Spleen 5×8 +
  фолбэк Unifont), reflow с сентинелами переноса `↵`, 1-битный атлас в
  продакшне (измерено лучше, чем AA, на Fable).
- `src/core/render-cache.ts` — LRU-мемоизация детерминированных рендеров
  (иначе статичный slab + замороженные фрагменты истории перерисовывались бы
  на каждом запросе).
- `src/core/history.ts` — сворачивает старые ходы в append-only замороженные
  фрагменты изображений, которые остаются побайтово идентичными, чтобы
  кэширование промптов продолжало срабатывать.
- `src/core/png.ts` — минимальный детерминированный PNG-энкодер (без
  нативных зависимостей).

## Защитные механизмы

- Allowlist моделей (`src/core/applicability.ts`): в изображение переводятся
  только модели, прошедшие бенчмарк чтения; всё остальное проходит побайтово
  неизменным.
- Побайтово точные значения (SHA, id) едут как текст в факт-листе рядом с
  изображением (`src/core/factsheet.ts`); восстанавливаемые оригиналы через
  `emitRecoverable`.
- Нативные типизированные инструменты (`type !== 'custom'`) никогда не
  переписываются (гвард 400).
