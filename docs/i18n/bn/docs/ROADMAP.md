OpenAI/Gemini অডিট অফিসিয়াল ডকসের বিপরীতে, সম্পর্কিত টুলের বিশ্লেষণ,

6. ☐ **Gemini সাপোর্ট** (নতুন): `src/core/gemini.ts` + `gemini-model-profiles.ts` + প্রক্সিতে `:generateContent`/`:streamGenerateContent` রুট। নথিভুক্তযোগ্য জ্যামিতি: **1152×1536 (এক্সাক্ট ক্রপ ইউনিট 768, 4টি টাইল, 42.2 chars/token — 3টি প্রোভাইডারের মধ্যে সেরা নথিভুক্ত অনুপাত)**; ক্যালিব্রেট করার বাজি: `media_resolution:MEDIUM`-এ 768² (56.4) এবং Gemini 3 HIGH। সতর্কতা: Gemini-এর OpenAI-compat এন্ডপয়েন্ট ভুল বিলিং সহ OpenAI ট্রান্সফর্মারের মধ্য দিয়ে যেত।
