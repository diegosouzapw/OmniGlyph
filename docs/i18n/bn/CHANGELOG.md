## [1.0.0] — 2026-07-07

প্রথম পাবলিক রিলিজ।

### পণ্য

- **কনটেক্সট-অ্যাজ-ইমেজ কম্প্রেশন প্রক্সি**: প্রতিটি LLM রিকোয়েস্টের বাল্কি অংশগুলো
  (সিস্টেম প্রম্পট, টুল ডক্স, পুরনো ইতিহাস, বড় টুল আউটপুট) আপনার মেশিন থেকে বেরিয়ে
  যাওয়ার আগেই ঘন 1-bit PNG পৃষ্ঠায় পুনর্লিখন করে। লোকাল Node সার্ভার এবং
  Cloudflare Workers হোস্ট।
- **প্রতি-প্রোভাইডার এক্সাক্ট বিলিং গণিত** (`src/core/`): Anthropic 28px প্যাচ +
  3–4 টোকেন/ব্লক ওভারহেড (নিজস্ব সুইপ, শূন্য অবশিষ্টাংশ), OpenAI এবং Gemini
  ফর্মুলা অফিসিয়াল ডকসের বিপরীতে অডিট করা। প্যাকেজ রুটে এক্সপোর্ট করা হয়েছে
  (`anthropicImageTokens`, `resolveAnthropicVisionTier`, টিয়ার ক্যাপ)।
- **পরিমাপকৃত প্রোডাকশন রেন্ডার কনফিগ**: ঘন 1-bit গ্লিফ অ্যাটলাস (কোনো
  অ্যান্টি-অ্যালিয়াসিং নেই), স্ট্যান্ডার্ড-টিয়ার পৃষ্ঠা — প্রতিটি সিদ্ধান্ত
  `benchmarks/*/results/`-এ একটি বেঞ্চমার্ক রসিদ দ্বারা সমর্থিত।
- **বেঞ্চমার্ক হার্নেস** (`benchmarks/`): billing-sweep (টোকেন অ্যাকাউন্টিং) এবং
  density-frontier (মডেল/ডেনসিটি জুড়ে রিড-অ্যাকুরেসি ফ্রন্টিয়ার), API,
  OpenRouter, Claude Code CLI, অথবা OmniRoute (`--via-omniroute`) মাধ্যমে
  পুনরায় চালানো যায়।
- **রিফিউজাল রিট্রাই**: SSE/JSON স্নিফার যখন কোনো মডেল রেন্ডার করা পৃষ্ঠা প্রত্যাখ্যান
  করে তখন মূল রিকোয়েস্ট রিপ্লে করে (কিল সুইচ `retryRefusalWithOriginal`)।
- ডিটারমিনিস্টিক পৃষ্ঠার জন্য **LRU রেন্ডার ক্যাশ**।
- **OmniRoute ইঞ্জিন**: [OmniRoute](https://github.com/diegosouzapw/OmniRoute)-এ
  `omniglyph` কম্প্রেশন ইঞ্জিন হিসেবে চালু হয় (সিঙ্গেল মোড ও স্ট্যাকড পাইপলাইন),
  ফেইল-ক্লোজড গেট এবং ইমেজ-অ্যাওয়্যার টোকেন অ্যাকাউন্টিংসহ।
