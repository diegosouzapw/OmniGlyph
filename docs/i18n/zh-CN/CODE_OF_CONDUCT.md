# 贡献者公约行为准则

## 我们的承诺

作为社区的成员、贡献者与领导者,我们承诺让参与我们社区的每个人都能免受骚扰,
无论年龄、体型、可见或不可见的残疾、种族、性征、性别认同与表达、经验水平、
教育程度、社会经济地位、国籍、外貌、种族、宗教信仰,或性取向与性别认同如何。

我们承诺以有助于建立开放、友善、多元、包容且健康的社区的方式行事和互动。

## 我们的准则

有助于为我们的社区营造积极环境的行为示例包括:

- 对他人表现出同理心与善意
- 尊重不同的意见、观点与经历
- 给予并优雅地接受建设性反馈
- 对我们的错误承担责任,向受影响的人道歉,并从中吸取经验
- 不仅关注对我们自身最有利的事,也关注对整个社区最有利的事

不可接受的行为示例包括:

- 使用与性有关的言语或图像,以及任何形式的性骚扰或性挑逗
- 挑衅、侮辱或贬损性言论,以及人身攻击或政治攻击
- 公开或私下的骚扰行为
- 未经他人明确许可,发布他人的私人信息,例如实际地址或电子邮件地址
- 其他在专业场合中可被合理认定为不当的行为

## 执行责任

社区领导者负责厘清并执行我们可接受行为的准则,并将针对任何他们认为不当、
威胁、冒犯或有害的行为采取恰当而公正的纠正措施。

社区领导者有权利也有责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、
wiki 编辑、议题及其他贡献,并在适当时说明其审核决定的理由。

## 适用范围

本行为准则适用于所有社区空间,也适用于个人在公共场合正式代表社区时的行为。
代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体账号发帖,
或在线上或线下活动中担任指定代表。

## 执行

滥用、骚扰或其他不可接受行为的事件,可通过在以下地址开启私密的安全公告进行报告:
<https://github.com/diegosouzapw/OmniGlyph/security/advisories/new>
或直接发送邮件给维护者:diegosouza.pw@outlook.com。
对于涉及安全的事件,请参阅 [`SECURITY.md`](SECURITY.md)。
所有投诉都将得到及时且公正的审查与调查。

所有社区领导者都有义务尊重事件报告者的隐私与安全。

## 执行准则

社区领导者在判定任何违反本行为准则的行为所应承担的后果时,将遵循以下
社区影响准则:

### 1. 更正

**社区影响**:使用了不恰当的语言,或表现出被社区认为不专业或不受欢迎的其他行为。

**后果**:社区领导者发出的私下书面警告,说明违规行为的性质,
并解释该行为为何不当。可能要求当事人公开道歉。

### 2. 警告

**社区影响**:因单一事件或一系列行为而构成的违规。

**后果**:警告,并说明若继续该行为将产生的后果。在指定时间内,
不得与相关人员互动,包括不得主动联系负责执行行为准则的人员。
这包括避免在社区空间及外部渠道(如社交媒体)中的互动。违反这些条款
可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**:严重违反社区准则,包括持续的不当行为。

**后果**:在指定时间内,临时禁止与社区进行任何形式的互动或公开交流。
在此期间,不得与相关人员进行任何公开或私下互动,包括不得主动联系
负责执行行为准则的人员。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**:表现出持续违反社区准则的行为模式,包括持续的不当行为、
对个人的骚扰,或对特定群体的攻击或贬损。

**后果**:永久禁止在社区内进行任何形式的公开互动。
