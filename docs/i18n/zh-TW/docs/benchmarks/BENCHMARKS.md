**每區塊 +3(Fable 5)/ +4(Sonnet 4.5)**——所有列的殘差均為零。

## 2. 讀取準確率(density-frontier,十六進位/camelCase/數字需求 + 干擾項)

## 附錄 2026-07-06(3)——Gemini 2.5-flash(部分:免費檔位配額於執行中耗盡)
