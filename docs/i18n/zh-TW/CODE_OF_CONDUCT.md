# 貢獻者公約行為準則

## 我們的承諾

身為社群成員、貢獻者與領導者,我們承諾讓每個人都能在不受騷擾的環境中
參與我們的社群,無論年齡、體型、可見或不可見的身心障礙、族裔、性徵、
性別認同與表達、經驗程度、教育程度、社經地位、國籍、外貌、種族、
宗教信仰或性傾向與性認同為何。

我們承諾以有助於營造開放、友善、多元、包容且健康社群的方式行事與互動。

## 我們的標準

有助於為我們的社群營造正向環境的行為範例包括:

- 對他人展現同理心與善意
- 尊重不同的意見、觀點與經驗
- 給予並優雅地接受建設性回饋
- 對我們的錯誤承擔責任、向受影響的人道歉,並從中學習
- 專注於不只對個人最有利,而是對整個社群最有利的事物

不可接受的行為範例包括:

- 使用與性有關的語言或影像,以及任何形式的性關注或挑逗
- 酸言酸語、侮辱或貶損性言論,以及人身或政治攻擊
- 公開或私下的騷擾行為
- 未經明確許可,公開他人的私人資訊,例如實體地址或電子郵件地址
- 在專業場合中可被合理視為不當的其他行為

## 執行責任

社群領導者負責澄清並執行我們可接受行為的標準,並會針對任何他們認為
不當、具威脅性、冒犯性或有害的行為採取適當且公正的糾正措施。

社群領導者有權利也有責任移除、編輯或拒絕不符合本行為準則的評論、
提交紀錄、程式碼、wiki 編輯、議題及其他貢獻,並會在適當時說明
其審核決定的理由。

## 適用範圍

本行為準則適用於所有社群空間,也適用於個人在公開場合正式代表本社群時。
代表本社群的範例包括使用官方電子郵件地址、透過官方社群媒體帳號發文,
或在線上或線下活動中擔任指定代表。

## 執行方式

濫用、騷擾或其他不可接受的行為,可透過在下列網址開立私人安全通報回報
給負責執行的社群領導者:
<https://github.com/diegosouzapw/OmniGlyph/security/advisories/new>
或直接寄信給維護者 diegosouza.pw@outlook.com。
如涉及安全性相關事件,請參閱 [`SECURITY.md`](SECURITY.md)。
所有投訴都將被及時且公正地審查與調查。

所有社群領導者都有義務尊重事件回報者的隱私與安全。

## 執行準則

社群領導者在判定任何被視為違反本行為準則的行為之後果時,
將遵循以下社群影響準則:

### 1. 更正

**社群影響**:使用不當語言,或在社群中出現其他被視為不專業或
不受歡迎的行為。

**後果**:社群領導者發出私下的書面警告,說明違規行為的性質,
並解釋該行為為何不當。可能會要求公開道歉。

### 2. 警告

**社群影響**:單一事件或一連串行為所構成的違規。

**後果**:發出警告,並對持續的行為附帶後果。在指定期間內不得與
相關人員互動,包括對負責執行行為準則者的主動互動。這包括避免在
社群空間及社群媒體等外部管道中的互動。違反這些條款可能導致
暫時或永久封禁。

### 3. 暫時封禁

**社群影響**:嚴重違反社群標準,包括持續的不當行為。

**後果**:在指定期間內暫時禁止與社群進行任何形式的互動或公開溝通。
在此期間內,不允許與相關人員有任何公開或私下的互動,包括對負責
執行行為準則者的主動互動。違反這些條款可能導致永久封禁。

### 4. 永久封禁

**社群影響**:展現出違反社群標準的行為模式,包括持續的不當行為、
騷擾個人,或對特定群體的攻擊或貶損。

**後果**:永久禁止在社群內進行任何形式的公開互動。
