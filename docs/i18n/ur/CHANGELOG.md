# تبدیلیوں کا لاگ

فارمیٹ: [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) · سیمانٹک ورژننگ۔

## [1.0.0] — 2026-07-07

پہلا عوامی ریلیز۔

### پروڈکٹ

- **سیاق و سباق-بطور-تصویر کمپریشن پراکسی**: ہر LLM درخواست کے بھاری حصوں
  (سسٹم پرامپٹ، ٹول دستاویزات، پرانی تاریخ، بڑے ٹول آؤٹ پٹس) کو آپ کی مشین
  سے نکلنے سے پہلے گھنے 1-بٹ PNG صفحات میں دوبارہ لکھتا ہے۔ لوکل Node سرور
  اور Cloudflare Workers ہوسٹ۔
- **فراہم کنندہ کے مطابق عین بلنگ ریاضی** (`src/core/`): Anthropic 28px پیچز +
  3–4 ٹوکنز/بلاک اوور ہیڈ (اپنی سویپ، صفر بقیہ)، OpenAI اور Gemini
  فارمولوں کی سرکاری دستاویزات کے خلاف آڈٹ کی گئی۔ پیکیج روٹ پر ایکسپورٹ کی گئی
  (`anthropicImageTokens`, `resolveAnthropicVisionTier`, درجہ کی حدیں)۔
- **پیمائش شدہ پروڈکشن رینڈر کنفیگ**: گھنا 1-بٹ گلف ایٹلس (کوئی اینٹی-ایلیازنگ نہیں)،
  اسٹینڈرڈ-درجے کے صفحات — ہر انتخاب کو `benchmarks/*/results/` میں ایک بینچ مارک
  رسید کی حمایت حاصل ہے۔
- **بینچ مارک ہارنسز** (`benchmarks/`): billing-sweep (ٹوکن اکاؤنٹنگ) اور
  density-frontier (ماڈلز/کثافتوں میں پڑھنے کی درستگی کا فرنٹیئر)، جو
  API، OpenRouter، Claude Code CLI، یا OmniRoute (`--via-omniroute`) کے
  ذریعے دوبارہ چلائی جا سکتی ہیں۔
- **انکار پر دوبارہ کوشش**: SSE/JSON سنیفر اصل درخواست کو دوبارہ چلاتا ہے
  جب کوئی ماڈل رینڈر شدہ صفحے سے انکار کرے (کِل سوئچ `retryRefusalWithOriginal`)۔
- **LRU رینڈر کیش** متعین صفحات کے لیے۔
- **OmniRoute انجن**: [OmniRoute](https://github.com/diegosouzapw/OmniRoute)
  میں `omniglyph` کمپریشن انجن کے طور پر شپ ہوتا ہے (سنگل موڈ اور
  اسٹیکڈ پائپ لائن)، fail-closed گیٹس اور امیج-آگاہ ٹوکن اکاؤنٹنگ کے ساتھ۔

### اعداد و شمار (سب دوبارہ پیدا کرنے کے قابل)

- نمونہ UI رینڈر: 1015 حروف → 438×120 PNG، 254 → 84 ٹوکنز (**66.9% بچت**)۔
- اسٹینڈرڈ صفحہ 1568×728 = 1456 امیج ٹوکنز، قطع نظر اس کے کہ وہ کتنا متن رکھتا ہے۔
- Claude پروڈکشن کثافت پر گھنے 1-بٹ صفحات کو 100% پر پڑھتا ہے؛ Opus 4.8
  10×16 پر 77–87% پڑھتا ہے۔

### منفی فیصلے (پیمائش شدہ، رائے نہیں)

- **ہائی-ریز درجہ ایک بلنگ کا جال ہے**: 1928² صفحہ WYSIWYG بل ہوتا ہے لیکن
  انکوڈر کو مکمل ریزولوشن نہیں ملتی — دونوں درجے اسٹینڈرڈ صفحات رینڈر کرتے ہیں۔
- **GPT-5.5 مسترد**: گھنی پٹی کی 0/60 ریڈز اور متن کے کنٹرول کے مقابلے میں
  ~40× کمپلیشن افراط زر۔
- **gpt-4o-mini کبھی امیج نہیں کیا گیا** (2833/5667 ٹوکن فرش اسے غیر منافع بخش بناتا ہے)۔
- **Gemini 2.5-flash من گھڑت جواب دیتا ہے** گریز کرنے کے بجائے گھنے صفحات پر
  (0/26) — ادا شدہ کوٹے کے دوبارہ ٹیسٹ کا منتظر۔
