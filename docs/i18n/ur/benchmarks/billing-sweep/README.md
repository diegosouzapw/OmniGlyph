# Anthropic ویژن-بلنگ سویپ

مفت `count_tokens` سویپ جو دو کھلے جیومیٹری سوالات کا فیصلہ کرتا ہے:

1. **فارمولا** — کیا API `ceil(w/28) × ceil(h/28)` پیچز کا بل کرتا ہے
   (موجودہ دستاویزات) یا ریٹائر شدہ `w·h/750`؟ پروب سیٹ دونوں کو فی
   قطار 25–180 ٹوکنز سے الگ کرتا ہے۔
2. **درجہ** — کیا `claude-fable-5` کو ہائی-ریزولوشن کیپس ملتی ہیں (لمبا
   کنارہ ≤ 2576 px، ≤ 4784 بصری ٹوکنز)؟ `page-old-1928x1928` قطار
   فیصلہ کن ہے: ≈ **4761** پیمائش شدہ کا مطلب ہے ہائی-ریز WYSIWYG (پرانا
   بڑا صفحہ آج کے 1568×728 سے ~3.3× زیادہ حروف فی امیج رکھتا ہے، اسی
   حروف/ٹوکن پر)؛ ≈ **1521** کا مطلب ہے اسٹینڈرڈ-درجہ resample، اور
   1568×728 درست رہتا ہے۔

سیاق و سباق: موجودہ 1568×728 صفحے کے پیچھے 2026-07-01 سویپ (پڑھنے کی
صلاحیت کا آڈٹ، 2026-07-01) کی پیمائش `claude-sonnet-4-5` پر کی گئی —
ایک اسٹینڈرڈ-درجہ ماڈل — جبکہ پروڈکشن Fable 5 کو ہدف بناتا ہے، جسے
ویژن دستاویزات ہائی-ریزولوشن درجے میں رکھتی ہیں۔ اس آڈٹ نے موجودہ
صفحے کو بھی 1460 ٹوکنز پر پیمائش کیا: پیچ فارمولے کے 1456 کے قریب تر
بمقابلہ /750 کے 1522، یہ اشارہ دیتے ہوئے کہ API پہلے ہی پیچ بلنگ پر
منتقل ہو چکا تھا۔

## چلائیں

```bash
pnpm run build                              # dist/ پیشگی شرط (تمام evals کی طرح)
node benchmarks/billing-sweep/run.mjs --dry-run   # صرف پیش گوئیاں، بغیر کلید کے، $0

ANTHROPIC_API_KEY=sk-... node benchmarks/billing-sweep/run.mjs \
  --models claude-fable-5,claude-sonnet-4-5 --probe-multi --probe-20plus
```

API کو **براہ راست** ہٹ کرنا ضروری ہے — کبھی بھی OmniGlyph پراکسی کے
ذریعے نہیں، جو باڈی کو تبدیل کر دے گا۔ `count_tokens` مفت ہے؛ مکمل
سویپ ~25 درخواستیں کرتا ہے۔

## آؤٹ پٹ پڑھنا

فی ماڈل، ہر پروب قطار پیمائش شدہ امیج ٹوکنز (امیج-کے-ساتھ منفی صرف-متن
بیس لائن) کو تمام چار پیش گوئیوں (`patch`/`legacy750` × `standard`/`highres`)
کے خلاف دکھاتی ہے؛ خلاصہ اوسط مطلق بقیہ کے ذریعے مفروضات کی درجہ بندی
کرتا ہے۔ `--probe-multi` فی امیج کیپ چیک کرتا ہے (2×1092² ≈ 2×1521)؛
`--probe-20plus` >20-تصاویر کا اصول چیک کرتا ہے (>2000 px والا کنارہ
مسترد ہونا چاہیے، ری سائز نہیں)۔ قطاریں `results/*.jsonl` میں جاتی ہیں؛
پیش گوئی کی ریاضی `formulas.mjs` میں رہتی ہے، `tests/billing-sweep-formulas.test.ts`
کے ذریعے پن شدہ۔

## فیصلے کے بعد

- پیچ فارمولا تصدیق شدہ → OmniGlyph PR #27 (عین ری سائز ترجمہ) کو پورٹ
  کریں اور `src/core/transform.ts` میں `ANTHROPIC_PIXELS_PER_TOKEN` گیٹ
  ریاضی کو ہم آہنگ کریں۔
- Fable پر ہائی-ریز درجہ تصدیق شدہ → فی درجہ صفحہ جیومیٹری دوبارہ
  متعارف کریں (Fable/Opus 4.8/Sonnet 5 کے لیے 1928×1928-کلاس صفحات،
  اسٹینڈرڈ کے لیے 1568×728)، بالکل اسی طرح جیسے GPT پاتھ پہلے ہی اپنا
  `GPT_MAX_HEIGHT_PX` رکھتا ہے۔
