# Журнал змін

Формат: [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) · семантичне версіонування.

## [1.0.0] — 2026-07-07

Перший публічний реліз.

### Продукт

- **Проксі стиснення "контекст як зображення"**: переписує громіздкі частини
  кожного LLM-запиту (системний промпт, документацію інструментів, стару
  історію, великі виводи інструментів) у щільні 1-бітні PNG-сторінки ще до
  того, як вони покинуть вашу машину. Локальний Node-сервер і хост
  Cloudflare Workers.
- **Точна тарифікація для кожного провайдера** (`src/core/`): патчі
  Anthropic 28 px + накладні витрати 3–4 токени/блок (власний sweep, нульовий
  залишок), формули OpenAI та Gemini перевірені проти офіційної документації.
  Експортується з кореня пакета (`anthropicImageTokens`,
  `resolveAnthropicVisionTier`, ліміти тарифів).
- **Виміряна продакшн-конфігурація рендерингу**: щільний 1-бітний атлас
  гліфів (без згладжування), сторінки стандартного тарифу — кожен вибір
  підкріплений підтвердженням бенчмарку в `benchmarks/*/results/`.
- **Тестові стенди бенчмарків** (`benchmarks/`): billing-sweep (облік
  токенів) і density-frontier (межа точності читання за моделями/щільністю),
  які можна перезапустити через API, OpenRouter, Claude Code CLI або через
  OmniRoute (`--via-omniroute`).
- **Повтор при відмові**: сніфер SSE/JSON повторює оригінальний запит, коли
  модель відмовляється від рендеровної сторінки (вимикач
  `retryRefusalWithOriginal`).
- **LRU-кеш рендерингу** для детермінованих сторінок.
- **Движок OmniRoute**: постачається як движок стиснення `omniglyph` у
  [OmniRoute](https://github.com/diegosouzapw/OmniRoute) (окремий режим і
  стековий конвеєр), з fail-closed гейтами й обліком токенів з урахуванням
  зображень.

### Цифри (усі відтворювані)

- Приклад рендерингу UI: 1015 символів → PNG 438×120, 254 → 84 токени
  (**зекономлено 66,9%**).
- Стандартна сторінка 1568×728 = 1456 токенів зображення незалежно від того,
  скільки тексту вона несе.
- Claude читає щільні 1-бітні сторінки на 100% за продакшн-щільності;
  Opus 4.8 читає 77–87% за 10×16.

### Негативні рішення (виміряні, а не думки)

- **Тариф високої роздільності — це пастка тарифікації**: сторінка 1928² 
  тарифікується WYSIWYG, але енкодер не отримує повної роздільності — обидва
  тарифи рендерять стандартні сторінки.
- **GPT-5.5 відхилено**: 0/60 прочитань щільної смуги і роздування завершення
  ~у 40× проти текстового контролю.
- **gpt-4o-mini ніколи не перетворюється на зображення** (поріг у
  2833/5667 токенів робить це неприбутковим).
- **Gemini 2.5-flash конфабулює** замість того, щоб утримуватися, на щільних
  сторінках (0/26) — очікує повторного тесту з платною квотою.
